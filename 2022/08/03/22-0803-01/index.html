添加图片插件或采用图床

添加图片插件hexo-image-link

npm install hexo-image-link --save

注意：这里并没有采用官方推荐的hexo-asset-image插件，采用hexo-asset-image并没有效果（有点奇怪，之前的hexo博客一直用的hexo-asset-image插件）

这里放一个可能的hexo-asset-image插件解决方案（未进行尝试）：https://blog.csdn.net/qq_42009500/article/details/118788129

注意：不要同时安装hexo-asset-image插件和hexo-image-link插件，否则hexo-image-link插件会没效果

实际采用hexo-image-link来解决图片问题比hexo-asset-image更优，可以直接调整图片的大小，之前用hexo-asset-image无法调整图片大小，把图片宽度修改成400px之后，显示效果如下：

但该插件的缺点也很明显，插入的图片名称必须带有image（可能插件匹配规则就是匹配image字段），如果插入图片名称为1.png这种，则不会显示

图片居中显示

安装图片插件后，默认图片是在左边显示的，需要让图片居中显示

经过测试发现，只要让img标签外面包裹一层p标签即可让图片居中，我们可以在markdown文件中添加图片时，在图片前后手动加上<p>、</p>，但是这样太麻烦了，直接采用js即可

在themes\fluid\source\js下创建image.js，其内容如下：

$(function(){
	$('img').wrap(function(){
		return '<p>';
	})
});

然后在themes\fluid\_config.yml文件中引入js文件

custom_js:
    - /js/image.js

图片缩放为80%

插入图片后，有时显示出的图片会有点模糊，如下：

如果缩放80%，则显示更加清晰，如下：

为img标签添加class属性，并让class=”suofang”，在themes\fluid\source\js\image.js下添加如下内容：

$(function(){
	$('img').prop("class","suofang");
});

然后在themes\fluid\source\css下创建suofang.css，内容如下：

.suofang {
  zoom:0.8;
}

注意：这里缩放用的zoom属性，经过测试，该属性只有在谷歌浏览器下有用，在火狐和ie下都没效果

然后在themes\fluid\_config.yml文件中引入css文件

custom_css:
   - /css/suofang.css
